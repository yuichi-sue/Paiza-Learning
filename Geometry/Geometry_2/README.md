Paiza練習問題「円板と長方形の共通部分の判定」  
url:https://paiza.jp/works/mondai/s_rank_level_up_problems/s_rank_level_up_problems__geometry_2  
  
以下のような円板と長方形が与えられます。  
円板は、座標 (x1, y1) を中心とする半径 r の円板となっています。  
長方形は、座標 (x2, y2) を左下の頂点とし、座標 (x3, y3) を右上の頂点とする長方形となっています。  
  
この円板と長方形をそれぞれ境界および内部を含む図形として考えるとき、これら 2 つの図形が共通部分を持つかどうかを判定してください。  
なお、円板と長方形が接している場合も共通部分を持つとみなします。
